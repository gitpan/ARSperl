ars_GetActiveLink(ctrl,name)

This function fetches information regarding a specified active link.
On success
A scalar pointer to the active link attributes.
On failure
undef.

Example: 

      $alink = ars_GetActiveLink($c, "AR-Assets-GetContactinfo");
