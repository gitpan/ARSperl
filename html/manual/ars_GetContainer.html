ars_GetContainer(ctrl,name)

This function fetches information regarding a specified container (Active Link Guide, Application, Packing List or Filter Guide).

On success
A scalar pointer to the container attributes.
On failure
undef.

Example:

      $alink = ars_GetContainer($c, "ARReport-BuildFieldList");
