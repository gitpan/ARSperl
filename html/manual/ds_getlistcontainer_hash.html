GetListContainer Hash Values

An array of references to this structure is returned by ars_GetListContainer. 

{
 "containerName"   => string,
 "containerType"   => string,
 "ownerObjList"    => reference to an array
}
See ars_GetListContainer.pl in the examples directory.
