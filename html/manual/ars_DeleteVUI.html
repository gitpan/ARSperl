ars_DeleteVUI(ctrl, schema, vuiId)

Delete the given VUI (specified by Id) associated with the specified schema from the server specified in the given control structure. Requires Administrator privs to use.

On success
Returns 1.
On failure
Returns 0.

Example: 

      ars_DeleteVUI($ctrl, "HD:HelpDesk", 192837465) || 
		die "$ars_errstr";
ars_DeleteVUI was introduced in version 1.5 of ARSperl.
