GetVUI Hash Values

An array of references to this structure is returned by ars_GetVUI. 

{
 "schema"             => string,
 "vuiId"              => integer,
 "vuiName"	      => string,
 "owner"              => string,
 "helpText"           => string,
 "lastChanged"        => string,
 "changeDiary"        => string,
 "timestamp"          => integer,
 "props"              => reference to an array of props 
}
See the ar.h file that accompanies the ARSystem API for a lengthy and well commented list of properties and property values.
