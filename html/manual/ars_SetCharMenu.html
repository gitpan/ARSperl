ars_SetCharMenu(ctrl, name, menuHash)

This function modifies an existing menu. You must first populate the menuHash before calling this routine.

On success
Returns 1
On failure
Returns 0

Example:

This example changes the name and the refresh code of the menu "TEST:CharMenu". 
      ars_SetCharMenu( $c, "TEST:CharMenu, {
        name        => "TEST:CharMenu2",
        refreshCode => "open",
      } ) || die $ars_errstr;

This function was introduced in version 1.90 of ARSperl
