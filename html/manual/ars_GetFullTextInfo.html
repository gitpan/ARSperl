ars_GetFullTextInfo(ctrl)

Fetch information about the current Full Text Search configuration on the server. See also: GetFullTextInfo Hash Values.

On success
Returns a HASH.
On failure
Returns undef.

Example: 

	($a = ars_GetFullTextInfo($ctrl)) || 
		die "$ars_errstr";
ars_GetFullTextInfo was introduced in version 1.5 of ARSperl
