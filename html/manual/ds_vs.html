Value Structure

ARValueStruct is just converted to the associated perl type, with the following exceptions:

Keyword
Keywords are denoted by a string surrounded by nulls. For example, $TIMESTAMP$ is converted to the perl string "\0timestamp\0". C doesn't allow nulls in strings so this doesn't introduce any ambiguity.
Byte lists
Byte lists are translated into a Byte List Hash.
Diary String
A Diary string is translated into a Diary Hash.
Coordinate List
A Coordinate List is translated into a Coord Hash.
