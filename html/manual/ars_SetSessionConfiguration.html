ars_SetSessionConfiguration(ctrl, ...)

Set specified client session configuration.
On success
Returns 1
On failure
Returns 0

Example: 

	      ars_SetSessionConfiguration( $ctrl, 
                                &ARS::AR_SESS_CLIENT_TYPE, 7000 )
			|| die "$ars_errstr";

ars_SetSessionConfiguration was introduced in version 1.90 of ARSperl
