ars_DeleteEntry(ctrl,schema,entry_id)

This function deletes the entry (row) in the schema matching entry_id. Don't ask why it returns 0 on success.

On success
Returns 0.
On failure
Returns -1.

Example: 

      if (ars_DeleteEntry(ctrl,"Schema01",$entry_id) == -1) {
	die $ars_errstr;
      }
