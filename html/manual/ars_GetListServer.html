ars_GetListServer()

This function determines what servers are available from the machine that the script is running on.

On success
Returns an array of one or more servers as defined in the /etc/ar file.
On failure
Returns undef.

Example: 

      @servers = ars_GetListServer();
      for($i = 0; $i <= $#servers ; $i++) {
           print "$servers[$i]\n";
      }
