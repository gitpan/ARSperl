GetListGroup Hash Values

This structure is returned by ars_GetListGroup. 

{
 "groupId"      => reference to an array of integers,
 "groupType"    => reference to an array of integers,
 "groupName"    => reference to an array of array references
}
Each array reference contained in the array referenced by groupName contains a list a strings.
