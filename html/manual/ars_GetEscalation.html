ars_GetEscalation(ctrl, name)

Fetch information about the named Escalation. See also: GetEscalation Hash Values.

On success
Returns a HASH.
On failure
Returns undef.

Example: 

	($a = ars_GetEscalation($ctrl, "ES-FooBar")) || 
		die "$ars_errstr";
ars_GetEscalation was introduced in version 1.5 of ARSperl
