ars_VerifyUser(ctrl)

This function fetches information regarding the logged in user.
On success
returns a reference to a HASH
On failure
undef.

Example: 

 my $vu = ars_VerifyUser($ctrl) || die "ars_VerifyUser: $ars_errstr" ;
 
 if ($vu->{adminFlag}) {print "You have administrative privs!\n";}
 else {print "You have NO administrative privs.\n";}
