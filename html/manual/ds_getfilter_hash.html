GetFilter Hash Values

This structure is returned by ars_GetFilter.

For ARS version 4.5: 

{
 "actionList"   => list of references to Action Hash
 "changeDiary"  => list of references to Diary Entry Structures,
 "elseList"     => list of references to Action Hash
 "enable"       => integer boolean,
 "helpText"     => string,
["lastChanged"  => integer,]
 "name"         => string,
 "opSet"        => integer bitmask,
 "order"        => integer,
 "owner"        => string,
 "query"        => reference to internal struct,
 "schemaList"   => list of one or more strings,
 "timestamp"    => integer
 "objPropList"     => list of references to Property Hash
}
For ARS version 3.x and 4.0.x: 
{
 "actionList"   => list of references to Action Hash
 "changeDiary"  => list of references to Diary Entry Structures,
 "elseList"     => list of references to Action Hash
 "enable"       => integer boolean,
 "helpText"     => string,
["lastChanged"  => integer,]
 "name"         => string,
 "opSet"        => integer bitmask,
 "order"        => integer,
 "owner"        => string,
 "query"        => reference to internal struct,
 "schema"       => string,
 "timestamp"    => integer
}
For ARS 2.1 and earlier: 
{
 "actionList"   => list of references to Action Hash
 "changeDiary"  => list of references to Diary Entry Structures,
 "enable"       => integer boolean,
 "helpText"     => string,
["lastChanged"  => integer,]
 "name"         => string,
 "opSet"        => integer bitmask,
 "order"        => integer,
 "owner"        => string,
 "query"        => reference to internal struct,
 "schema"       => string,
 "timestamp"    => integer
}
