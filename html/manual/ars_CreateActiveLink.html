ars_CreateActiveLink(ctrl, activeLinkHash)

This function creates an new active link. You must first populate the activeLinkHash before calling this routine.

activeLinkHash is an Active Link Attributes hash.

On success
Returns 1
On failure
Returns 0

Example:

This example copies an existing active link to a new active link. I.e. it simply re-saves the active link under a new name. 
      ($al = ars_GetActiveLink($c, "AL-foobar")) ||
	    die $ars_errstr;
      $al->{name} = "Copy of ".$al->{name};
      ars_CreateActiveLink($c, $al) || die $ars_errstr;
Notes: Hash keys helpText, owner and changeDiary are optional when calling this routine.

This function was introduced in version 1.50 of ARSperl
