GetFullTextInfo Hash Values

This structure is returned by ars_GetFullTextInfo. 

{
 "StopWords"     => reference to an array of strings,
 "CaseSensitive" => integer,
 "State"         => integer,
 "CollectionDir" => string,
 "MatchOp"       => integer
}
CaseSensitive
0 = Insensitive, 1 = Sensitive
MatchOp
0 = Force Leading and Trailing
1 = Ignore Leading and Force Trailing
2 = Ignore Leading
3 = Remove Loading and Trailing
4 = Query Unchanged
State
0 = Disabled 1 = Enabled
