ars_GetListGroup(ctrl, userName=NULL)

Fetch information about the available groups on a server, optionally restricting the list to a particular user name.

See also: GetListGroup Hash Values.

On success
Returns a HASH.
On failure
Returns undef.

Example: 

	($a = ars_GetListGroup($ctrl)) || 
		die "$ars_errstr";
ars_GetListGroup was introduced in version 1.5 of ARSperl
