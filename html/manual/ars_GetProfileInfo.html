ars_GetProfileInfo(ctrl)

Returns a HASH consisting of ARSperl profiling statistics.

Here is an example:

	%profile = ars_GetServerStatistics($ctrl);

	print "startTime= ".$profile->{startTime}." queries=".$profile->{queries}."\n";
On success
returns a HASH
On failure
returns undef
