ars_GetClientCharSet(control)

Retrieves a string that represents the name of the character set the client is using. The API assumes that all character data the client passes it is encoded in this character set, and returns all character data encoded in this character set. If this differs from the server charset (see ars_GetServerCharSet), the API converts the data to the right character set.
On success
Returns a string.
On failure
Returns undef.

Example: 

    my $charset = ars_GetClientCharSet($ctrl);
        print "This client is using character set $charset\n";

ars_GetClientCharSet was introduced in version 1.92 of ARSperl
