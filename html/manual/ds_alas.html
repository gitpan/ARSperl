Active Link Action Structure

{
 "macro"            => reference to Active Link Macro Structure,
      or 
 "assign_fields"    => reference to Field Assign Structure,
      or 
 "process"          => string,
      or 
 "message"          => Status Structure, (2.x - 3.x)
 "message"          => Message Structure, (4.x)
      or 
 "characteristics"  => Field Characteristics,
      or 
 "dde"              => not implemented,
      or 
 "none"             => undef
}
Notes: The message member of this hash was previously a string. As of ARSperl v1.50, this is no longer true.
