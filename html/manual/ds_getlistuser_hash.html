GetListUser Hash Values

An array of references to this structure is returned by ars_GetListUser. 

{
 "userName"           => string,
 "connectTime"        => integer,
 "lastAccess"         => integer,
 "defaultNotifyMech"  => integer,
 "emailAddr"          => string,
 "licenseTag"         => reference to an array,
 "licenseType"        => reference to an array,
 "currentLicenseType" => reference to an array
}
Note: We've never seen the server actually return values for emailAddr or defaultNotifyMech.

See ars_GetListUser.pl in the examples directory.
