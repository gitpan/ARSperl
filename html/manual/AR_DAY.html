ARS::AR_DAY(mask, dayNumber)

Determine if the specified day number is set in the mask. Days range from 1 to 31. Mask, depending upon semantics, might contain the days of the month (1 - 31) or the days of the week (1 - 7).

On success
Returns 1 (TRUE).
On failure
Returns 0 (FALSE).

Example: 

	if(ARS::AR_DAY($mask, 13) == 1) {
		print "The 13th is set.\n";
	}
ARS::AR_DAY was introduced in version 1.5 of ARSperl
