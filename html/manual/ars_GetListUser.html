ars_GetListUser(ctrl, userListType=AR_USER_LIST_MYSELF)

Retrieve information about users. Can be restricted to the current user (AR_USER_LIST_MYSELF), all registered users (AR_USER_LIST_REGISTERED) or currently attached users (AR_USER_LIST_CURRENT).

See also: GetListUser Hash Values.

On success
Returns an array of HASH references.
On failure
Returns undef.

Example: 

	(@a = ars_GetListUser($ctrl)) || 
		die "$ars_errstr";
ars_GetListUser was introduced in version 1.5 of ARSperl
