ars_Import(ctrl, importBuf, type, name, ...)

Import objects of type type and name name from the importBuf.
On success
Returns 1.
On failure
Returns 0.

Example: 

        $buf = `cat /tmp/importfile.arx`;
	($a = ars_Import($ctrl, $buf, "Schema", "HD:HelpDesk", 
			"Active_Link", "AL-FooBar")) || 
		die "$ars_errstr";
The above example will read an AR Export file (text file) into a scalar and will import Schemas and Active Links from that file into the ARSystem. Other object types are:
  • Schema
  • Schema_Defn
  • Schema_View
  • Schema_Mail
  • Filter
  • Active_Link
  • Admin_Ext
  • Char_Menu
  • Escalation
ars_Import was introduced in version 1.3 of ARSperl
